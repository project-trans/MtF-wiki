{{< notice warning >}}

全身麻酔時の喉頭挿管は声帯を刺激する可能性があるため、声帯手術後 1 年以内の全身麻酔手術は一般に推奨されません。性別適合手術を希望する場合は医師へ自分から伝えてください。医師が腰椎麻酔での手術を勧めることがあります。

{{< /notice >}}

{{< notice warning >}}

声帯手術の効果には個人差があり、重い副作用が起こる可能性もあります。副作用には声量低下、強い嗄声、術後の激しい咳による縫合部の離開などがあります。結果は満足できるものから変化が小さいものまであり、予測できません。手術を決める前に慎重に検討してください。

{{< /notice >}}

## 概説 {#overview}

声の女性化手術の前には、一般に次の確認が必要です。

1. {{< wiki 感染症 >}} 4 項目
1. 音声検査
   検査項目：高音/低音、音域、発声方法、最長発声持続時間、基本周波数など
1. [喉頭鏡検査](https://ja.wikipedia.org/wiki/%E5%96%89%E9%A0%AD%E9%8F%A1)

原則として、先に [声の女性化トレーニング]({{< ref "vocal-therapy-overview" >}}) を行うことが勧められます。声のトレーニングは発声方法や語用など、基本周波数だけでは決まらない部分を変えることができます。

声の女性化手術で変えられるのは主に発声周波数であり、声の大きさは程度の差はあれ低下します。

自分の声が期待に達しない具体的な理由が分からない場合は、基本周波数を自分で測るか、声のトレーニングに対応できる医療機関で音声評価を受けることができます。

日常生活の習慣も声に影響します。以下はいずれも長期・大量の場合に影響しやすい要因です。

- 喫煙（電子タバコ、受動喫煙を含む）
  - 喫煙は声帯の腫れを引き起こし、声へ大きく影響します。
- 飲酒
- コーヒー
- 夜更かし
- 就寝前 2-3 時間以内の食事、または大量の飲水
  - 胃酸逆流を起こし、声帯を刺激する可能性があります。
- 刺激物の摂取
  - 唐辛子、酸性食品、氷水など。

{{< notice info >}}

手術は万能ではありません。術後も、発音や話し方の印象などを声のトレーニングで調整する必要があります。

{{< /notice >}}

中国国内の病院では現在、内視鏡下の [声帯短縮術]({{< ref "#method" >}}) が主流です。輪状甲状軟骨接近術は、声帯短縮術に適さない症例、または [喉仏切除]({{< ref "tracheal-shave" >}}) と同時に輪状甲状軟骨接近術を希望する症例で主に用いられます。

一般に術後 3 か月は刺激物を避ける必要があります（主治医の指示に従ってください）。咳は術後の縫合離開につながることがあります。そのため、病院によっては咳や発声を避ける目的で術中に薬剤を追加することがあります。

術後に {{< wiki 嗄声 >}} や声調低下が出る場合は、比較的重い術後合併症です。

## 要件 {#prerequirements}

> 要件は病院ごとに異なります。全体として、中国国内の声の手術の前提条件は SRS よりかなり緩いことが多いですが、具体的には診察時の説明に従ってください。

## 術式 {#method}

```csv
術式,方法,長所,短所
輪状甲状軟骨接近術<br>輪状軟骨固定術,高音を出すときには輪状甲状筋が収縮し、<br>輪状軟骨と甲状軟骨が近づいて声帯が引き伸ばされる。<br>そのため甲状軟骨と輪状軟骨を近づけることで声帯を緊張させる,喉仏も同時に処理できる,可逆性がある一方、<br>再手術が必要になるリスクがある。
声帯短縮術,喉頭鏡下で声帯を短く縫合する,安定しており低侵襲で外見上の傷跡がない,不可逆、声量低下、効果にばらつきがある
甲状軟骨声帯縮小術<br>声帯壁縮小術,甲状軟骨の前半部を切除し、<br>切除部を用いて声帯を短縮する,N/A,N/A
レーザー音声手術,喉頭鏡下でレーザーを用い、声帯振動に最も関与する<br>声帯後半部の粘膜を蒸散する,N/A,N/A
声帯横隔膜形成術,N/A,N/A,N/A
```

## 声に影響する要因 {#key-factor}

発話基本周波数、共鳴、抑揚、声質、明瞭度、発話強度、発話速度、韻律。

### 声帯手術に適さない可能性がある状況 {#not-suitable}

{{< notice warning "注意" >}}
以下は受診時に医師から手術に適さないと説明された状況に基づくものです。実際に手術するかどうかは、最終的には医師と患者双方の判断によります。
{{< /notice >}}

- 先天性声帯発育不全
- 声帯閉鎖不全
- 声門閉鎖不全
- 声帯の過度な弛緩
- 発声パターンが単一すぎる
