通院が難しい地域に住んでいる方や、対面受診のハードルが高い方向けに、**オンライン診療または郵送処方による HRT** に対応しているクリニックをまとめます。

すべての情報は各クリニック公式サイトに基づきますが、料金・対応範囲は変更されることがあるため、受診前に必ず公式サイト・電話で最新情報を確認してください。

{{< notice warning >}}

オンライン処方であっても、初回診察・血液検査の対面受診を求められることがあります。また、自己注射の同意書サインや指導は対面で行うのが一般的です。

{{< /notice >}}

## オンライン処方・郵送対応クリニック

### 恵比寿TGクリニック（東京都渋谷区）

- 公式：<https://ebisutgclinic.com/>
- 詳細：[恵比寿TGクリニック]({{< ref "/docs/hrt/tokyo/ebisutgclinic" >}})
- 流れ：来院または LINE ビデオ通話で初診 → 自己注射指導 → 二回目以降オンライン注文（薬剤を郵送）
- 全国の自己注射患者 2,500 名以上の実績

### 自由が丘 MC クリニック（東京都目黒区）

- 公式：<https://www.gid-mcclinic.com/>
- 詳細：[自由が丘 MC クリニック]({{< ref "/docs/hrt/tokyo/mc" >}})
- 性別不合（GI）学会認定医による電話・Zoom 等のオンライン診療
- 遠方患者向けにカウンセリング＋ホルモン療法の継続管理に対応

### 女性医療クリニック LUNA 横浜元町（神奈川県横浜市）

- 公式：<https://www.luna-clinic.jp/transgender/online/>
- 詳細：[LUNA 横浜元町]({{< ref "/docs/hrt/kanagawa/luna" >}})
- 「トランスジェンダー＆ジェンダーダイバース外来」専科でオンライン処方対応
- 再診を中心に、性に関する相談・妊孕性カウンセリング枠もオンラインで提供

### 宮川クリニック（愛知県丹羽郡扶桑町・新栄宮川クリニック）

- 公式：<https://miyakawaclinic.com/gd/>
- 詳細：[宮川クリニック]({{< ref "/docs/hrt/aichi/miyakawa" >}})
- 「北海道から石垣島まで」を謳う全国対応のオンライン診療
- 性別変更申立書（診断書）作成にも対応

### 九州ベテルクリニック福岡（福岡県福岡市博多区）

- 公式：<https://gid.bethel.clinic/>
- 詳細：[九州ベテルクリニック福岡]({{< ref "/docs/hrt/fukuoka/bethel" >}})
- コンシェルジュ型クリニック（電話予約不可、オンライン申込みのみ）
- 大学病院での予約待ち中でも並行してホルモン治療を開始可能

## 留意事項

- **保険適用は対面通院が原則**。オンライン処方は基本的に自費診療となる
- ホルモン剤の自己注射には**同意書のサインと注射手技指導**が前提
- 採血モニタリング（半年〜1 年ごと）は近隣の検査機関で受け、結果を主治医に共有する形が一般的
- 一部クリニックは「他院で適正に HRT を受けていた方」のみオンライン継続を受け入れる方針
