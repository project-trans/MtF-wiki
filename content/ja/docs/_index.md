{{< mtf-wiki >}} へようこそ！ここは、一部のトランス女性やノンバイナリー／ジェンダーにとらわれない方々の助けになれるかもしれない場所です。

{{< notice info >}}
本ウィキは当初、中国のトランスジェンダーの相互扶助と福祉のために設立されたもので、後に日本向けのページを開設しました。そのため、翻訳は全体的に未完成です（ご了承ください）。特に医療系の情報などについては、[中国版](https://mtf.wiki/zh-cn/)からの翻訳が多数必要です。お手伝いいただける場合は、心より感謝いたします。

翻訳を含むご投稿は <mtfwiki@project-trans.org> までお送りください。GitHub が使える方は[公式リポジトリ](https://github.com/project-trans/MtF-wiki) への PR でも構いません。翻訳やお住まいの地域のトランスに関する情報のご提供をお待ちしております。
{{< /notice >}}

MtF（Male-to-Female）、トランスジェンダー女性とは、[出生時に割り当てられた性別](https://lgbtq.fandom.com/ja/wiki/%E5%87%BA%E7%94%9F%E6%99%82%E3%81%AB%E5%89%B2%E3%82%8A%E5%BD%93%E3%81%A6%E3%82%89%E3%82%8C%E3%81%9F%E6%80%A7%E5%88%A5)が男性である女性のことである。

そのほかにも、さまざまな方が女性的な身体表現を目指し、性別移行を行っています。
{{< mtf-wiki >}} という名前ですが、たとえ性別自認が女性でなくても、女性的な性別移行に挑んでいる全ての方を応援しています。

{{< mtf-wiki >}} は、女性傾向のトランスジェンダーに関する知識統合サイトとして、医療情報や各種ガイドを知ることができる、無料でオープンかつ継続的に更新される場を目指しています。

現在の {{< mtf-wiki >}} 日本語版はまだ始まったばかりで、資料が不十分だったり、誤りが残っていたりするかもしれません。熱意あるコミュニティの皆さまの参加と貢献を歓迎します。

このサイトを作った初めの思いは、多くのトランス女性に知識の場を提供すること、そして性別移行という選択がより肯定されることへの願いでした。

執筆の過程で多くの資料を参考にしました。資料の作者様にも感謝いたします。
