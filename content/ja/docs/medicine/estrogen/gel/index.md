エストラジオールゲルは皮膚から吸収されます。使いやすく、肝初回通過効果を完全に避けられるため、比較的安全性が高い投与経路です。ただし全体的な費用は高めです。

専門家コンセンサス：[雌二醇凝胶临床应用指导建议](http://medi-guide.meditool.cn/ymtpdf/E60E60E5-CF7F-0A27-75B3-BE7B935D762D.pdf)[^4]

## 使用方法と用量 {#usage}

{{< notice info "ガイド" >}}
{{< expand "Oestrogel 用法・用量説明カード" >}}
![Oestrogel dosage card](/images/medicine/gel/estradiol-gel-intro.jpg)
{{< /expand >}}
動画：[爱思妥用药指导](https://www.bilibili.com/video/BV1f44y1D7To)
{{< /notice >}}

- **使用時期：** 朝または夜の使用が推奨されます[^5]。
- **用法：外用。** 腕、腹部、臀部、大腿内側など、吸収しやすく日常生活を妨げにくい部位に塗布します[^1]。これらの部位の吸収率はおおむね同等です[^12]。擦り込む必要はなく、乾燥後は衣服を汚しません。
- **用量：** 1 日 1～2 計量尺（2～4 ポンプ）、すなわち 2.5～5 g のゲル[^5]。1 計量尺（2.5 g/日）では、定常状態のエストラジオール血漿濃度は平均 60～80 pg/mL です[^1] [^5] [^7]。
- **搾り出し方：** チューブ開口部を完全に開け、計量尺の溝の端から反対側までゆっくり連続して絞り、ゲルを円柱状に満たします。

**注意：**

- **粘膜に塗らないでください。** アルコール刺激を起こすことがあります。
- **乳房周囲に塗らないでください。** {{< tag/neg "乳がんを誘発する可能性があります。" >}} [^1] [^2] [^5]
- ゲルはアルコール（エタノール）を含みます。傷のある部位を避け、乾くまでは火気から遠ざけてください[^2]。
- 塗布後は乾いてから服を着てください[^2]。
- 使用後 1 時間以上は、水濡れや日焼け止めなどを避けてください[^2] [^6]。
- 採血前に採血部位の近く（上腕など）へ塗布すると、エストラジオール検査値が偽高値になることがあります。
- 効果が強すぎる、または弱すぎる場合は、医師に相談して用量を調整してください。

## 副作用[^1] [^2] [^7]

- **静脈血栓塞栓症** {{< tag/pos "経口錠より大幅に低い" >}}[^3]
- 悪心、下痢、腹痛、腹部膨満などの消化器症状
- 発疹、かゆみ、塗布部位反応
- 乳房痛
- 体重変化
- 性機能低下、不妊
- 乳がんリスクの上昇
- 発熱などのインフルエンザ様症状

{{< notice warning "注意" >}}
過量使用では副作用が強く出る可能性があります。定期的にホルモン値を測定し、必要に応じて用量を調整してください。
{{< /notice >}}
