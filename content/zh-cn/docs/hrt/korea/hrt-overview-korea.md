## 概况

在韩国，跨性女性的女性化 HRT 通常以 **雌激素** 为基础，必要时再 **联合抗雄激素**。[^jmm-review][^single-center] 文献中报告的抗雄激素包含螺内酯、醋酸赛普龙（CPA）以及 GnRH 激动剂；雌激素则有口服、肌注与经皮凝胶等多种剂型。处方医师涵盖精神科·内分泌内科·泌尿外科·妇产科以及部分一般科诊所。[^namu] 注意全程 **不在国民健康保险覆盖范围**，从精神科诊断书、检查、药物到注射费用都需要本人自费。[^lgbt-health][^kihasa]

## 一般流程

不同医疗机构可将以下 3 步合并办理。

1. **精神科评估与诊断**
   - 通常需要 1~数次面谈和心理测验，由精神科医师开具 **F64.0（性别认同障碍）** 或 ICD-11 **HA60（Gender Incongruence）** 诊断书。
   - 建议提前确认机构是否对跨性别友好、能否出具诊断书。
   - 费用一般为：私立医院约 30 万韩元，大学附属医院 40~50 万韩元，均为非保险。[^namu-trans][^transroadmap]
2. **HRT 前检查**
   - 常规血液检查（肝功能、肾功能、电解质、血脂、血糖、CBC）。
   - 性激素六项（睾酮、雌二醇、LH、FSH、催乳素、SHBG）。
   - 有静脉血栓家族史、吸烟、肥胖等高风险者，应进行进一步评估。
3. **开始用药**
   - 持诊断书前往内分泌内科·泌尿外科·妇产科·一般科诊所，或在能"诊断 + 处方一站式"的诊所直接开始用药。
   - 与医师讨论选择口服 / 经皮 / 肌注的最适方案。

## 药物与用量

成人常用药物与剂量范围如下。[^jmm-review][^single-center]

### 雌激素

| 剂型 | 常用剂量 |
| :--- | :--- |
| 口服/舌下含服 戊酸雌二醇 / 雌二醇（如 Progynova） | 2~6 mg/日 |
| 经皮 雌二醇凝胶（如 Divigel） | 2~6 mg/日 |
| 肌注 戊酸雌二醇（如 Progynon Depot） | 每 1~2 周 5~20 mg 肌注 |

> 经皮制剂的 VTE（静脉血栓）风险相对较低。35 岁以上、吸烟、肥胖、高血压等高危人群通常优先选择经皮路径。

### 抗雄激素

| 药物 | 常用剂量 |
| :--- | :--- |
| 螺内酯（Spironolactone） | 100~300 mg/日 |
| 醋酸赛普龙（CPA） | 12.5~50 mg/日 |
| GnRH 激动剂（亮丙瑞林、戈舍瑞林等） | 1 或 3 个月 depot 注射 |

> 醋酸赛普龙（CPA）在高剂量、长期使用或累计暴露增加时，与脑膜瘤（meningioma）风险升高相关，使用剂量与时长应与医师谨慎讨论。[^cpa-meningioma]

### 辅助药物

- 微粉化孕酮（用于胸部发育辅助）的使用因医院而异，临床证据有限。
- 在抗雄激素充分起效后，通常无需常规使用非那雄胺/度他雄胺。

## 监测计划

标准建议为 **首年每 3 个月、之后每 6~12 个月** 评估一次。[^jmm-review][^wpath]

- 测量项目：血清睾酮、雌二醇、肝肾功能、血脂、电解质（使用螺内酯时关注血钾）、催乳素、血糖。
- 参考范围
  - 睾酮：常以"顺性别女性范围"或大致 < 50 ng/dL 作为参考。
  - 雌二醇：常以 100~200 pg/mL 作为参考，但肌注剂型的测量值会因抽血时间不同而显著差异，需结合临床表现解读。
- 自我评估：胸部发育、血压、体重、情绪变化等。

## 未成年人

- 韩国部分机构使用 **GnRH 激动剂** 进行青春期阻断，但提供机构有限，需监护人同意。[^single-center]
- 通常在 Tanner 2 期之后考虑使用青春期阻断剂。
- 成人剂量 HRT 在未满 19 周岁者中的使用，因机构政策与监护人同意而异，无统一标准。

## 费用

- HRT 全程自费。药物部分每月约 **3~15 万韩元**。[^namu-trans][^transroadmap]
- 精神科诊断书、血液检查、注射操作另计。
- 部分药物在以"其他适应症"（如更年期、高血压）开具时，可能进入保险，但前提是适应症成立，需医师判断。

## 外国人就医

- 一般凭外国人登录证或护照等身份证件即可在韩国医疗机构就医；若已加入韩国国民健康保险，请同时携带保险相关材料。
- 高丽大学安岩医院、江东圣心医院等设有国际诊疗中心，提供英文等语种服务。[^medical-avenue]
- 外国诊断书不一定被韩国医师认可，可能需要重新评估。
- 若需将韩国就医记录用于本国，建议申请英文版并办理公证·阿波斯蒂尔。

## 已知诊疗机构（示例）

仅为社群报告整理的示例，并非全部清单。务必直接电话/邮件确认。

### 综合·大学医院

- **东亚大学医院**（釜山）— 精神科·内分泌内科·整形外科联合。
- **江东圣心医院 LGBT+ 医疗中心**（首尔）— 多学科一站式诊疗。[^transmap]
- **高丽大学安岩医院**（首尔）— 国际诊疗中心提供 HRT 处方。[^medical-avenue]

### 独立诊所

- **金尚俊精神科医院**（首尔）— 跨性别诊断书报告较多。[^namu-trans]
- 此外部分妇产科·内分泌内科·一般科医生亦有处方案例。建议结合人权团体·社群信息事先确认。
